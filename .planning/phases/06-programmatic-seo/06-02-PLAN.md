**Create `src/app/[locale]/trade/import/[slug]/page.tsx`:**

Follow the exact pattern from `src/app/[locale]/sectors/[slug]/page.tsx` and `src/app/[locale]/blog/[slug]/page.tsx` but adapted for PSEO content from data files instead of translation files.

Structure:
1. `export const dynamicParams = true;` (ISR fallback -- different from sectors which uses `false`)
2. `export const revalidate = 86400;` (24-hour revalidation)
3. `generateStaticParams()` -- iterate `routing.locales` x `productPages` array. Return `{ locale, slug }` pairs.
4. `generateMetadata()` -- await params, look up page via `getProductPage(slug)`, return `getPageMetadata({ locale, path: '/trade/import/${slug}', title: t(page.meta.title, locale), description: t(page.meta.description, locale) })`. Return empty `{}` if page not found.
5. Default export async server component:
   - `await params` to get locale + slug
   - `setRequestLocale(locale)` from next-intl/server
   - Look up page via `getProductPage(slug)`, call `notFound()` if not found
   - Use `t(content, locale)` for all text rendering

Page layout sections (use existing design system components):
- **JsonLd** with `getBreadcrumbSchema` (Home > Trade > Import > [Product Title])
- **PageHero** with title, subtitle, backgroundImage from data
- **Breadcrumb** component below hero
- **Overview section**: Render each paragraph from `content.overview[]` in a prose-style layout with `ScrollReveal`. Use 7/5 grid (text/image) like sector detail pages.
- **Key Facts section**: Render as a responsive grid of fact cards (3 cols on desktop, 1 on mobile). Each card shows label + value. Use mesh-warm background. Use colored top borders on cards (alternating primary, coral, emerald, gold).
- **Trade Advantages section**: 4-card grid like sector detail advantages. Each card has title + description. Use `ScrollReveal` with staggered delay.
- **Import Process section**: Numbered step cards (1-5) in a vertical timeline or horizontal stepper layout. Use amber-50 warm background for visual variety.
- **Quality Standards**: Single paragraph in a highlighted card or callout box.
- **CTASection** with product-specific CTA text, linking to `/contact`.

Styling must match the existing "Bold Mediterranean" design system:
- `heading-serif` for headings
- `rich-card` class for cards with hover lift
- Gradient accent lines under section headings (`bg-gradient-to-r from-[#0d7377] to-[#d4613c]`)
- `ScrollReveal` on all below-fold sections (NOT on hero -- per 04-03 decision)
- `mesh-warm` background on alternating sections
- Color-coded card borders using primary (#0d7377), coral (#d4613c), emerald (#2d8a6e), gold (#e8a840)
