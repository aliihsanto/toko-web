## Qu'est-ce que le code SH et pourquoi est-il si important ?

Le code du Système harmonisé (SH) est un code numérique unique attribué à chaque produit dans le commerce international. Ce code détermine comment un produit sera classé en douane, quels taux de droits lui seront appliqués, s'il est soumis à un contingent ou une interdiction, et quels documents sont requis. Un code SH erroné peut entraîner le paiement de droits injustifiés, des retards en douane et même des amendes administratives.

Développé par l'Organisation mondiale des douanes (OMD), le **Système harmonisé (SH)** est la norme de classification internationale utilisée par plus de 200 pays. En Turquie, le code GTİP est appliqué sur **12 chiffres** et constitue une version étendue du code SH international.

> En Turquie, environ **15 000** demandes de renseignements tarifaires contraignants sont déposées chaque année. Ce chiffre illustre la complexité et le caractère controversé de la classification.

## Structure du Système harmonisé (SH)

### Hiérarchie des codes

Le code SH possède une structure hiérarchique progressant des catégories générales vers des descriptions de produits spécifiques :

- **2 premiers chiffres (Chapitre)** : déterminent la catégorie générale du produit. Par exemple, Chapitre 61 = Vêtements en bonneterie
- **4 premiers chiffres (Position)** : définissent la sous-catégorie. Par exemple, 6109 = T-shirts en bonneterie
- **6 premiers chiffres (Sous-position)** : identifient le produit au niveau de la norme internationale. Par exemple, 6109.10 = T-shirts en bonneterie de coton
- **7e-8e chiffres (Nomenclature combinée UE)** : détails supplémentaires dans le cadre de l'Union douanière UE
- **9e-10e chiffres (Subdivision nationale)** : distinction statistique propre à la Turquie
- **11e-12e chiffres** : codes statistiques supplémentaires

### Structure des chapitres

Le Système harmonisé est organisé en **97 chapitres** et **21 sections**. Les sections sont généralement réparties comme suit :

- **Sections I-IV** (Chapitres 1-24) : Animaux vivants, produits végétaux, denrées alimentaires
- **Sections V-VII** (Chapitres 25-40) : Produits minéraux, produits chimiques, plastiques
- **Sections VIII-X** (Chapitres 41-49) : Cuir, bois, articles en papier
- **Sections XI-XII** (Chapitres 50-67) : Produits textiles, chaussures
- **Sections XIII-XV** (Chapitres 68-83) : Pierre, céramique, verre, métaux
- **Sections XVI-XVIII** (Chapitres 84-92) : Machines, appareils électriques, véhicules
- **Sections XIX-XXI** (Chapitres 93-97) : Armes, mobilier, objets d'art

## Méthodes de détermination du code SH

### Règles d'interprétation

Six règles fondamentales d'interprétation s'appliquent à la classification SH. Ces règles constituent les principes de base utilisés pour déterminer dans quelle position un produit doit être classé :

**Règle 1 — Titres des chapitres et sections** : La classification est effectuée en priorité selon les libellés des positions et les notes de chapitres/sections. Les titres n'ont qu'une valeur indicative et ne sont pas juridiquement contraignants.

**Règle 2** : Se subdivise en deux sous-règles :
- **(a)** Les articles incomplets ou non assemblés peuvent être classés comme les articles complets
- **(b)** La référence à un article en une matière donnée couvre également les mélanges partiellement constitués de cette matière

**Règle 3** : Pour les marchandises pouvant être classées dans plusieurs positions :
- **(a)** La description la plus spécifique est préférée
- **(b)** Pour les mélanges, compositions ou assortiments, la matière/l'élément qui confère le caractère essentiel est pris en compte
- **(c)** Si les critères ci-dessus sont insuffisants, la position dont le numéro d'ordre est le plus élevé s'applique

**Règle 4** : Les marchandises ne pouvant être classées dans aucune position sont classées dans celle correspondant aux articles les plus analogues.

**Règle 5** : Contient des règles spéciales pour les étuis, boîtes et emballages.

**Règle 6** : La classification en sous-positions suit les mêmes principes.

### Renseignement tarifaire contraignant (RTC)

Les importateurs ou exportateurs souhaitant déterminer de manière définitive le code SH correct d'un produit peuvent demander un **Renseignement tarifaire contraignant** auprès de l'administration douanière. Caractéristiques du RTC :

- La demande est déposée auprès de l'administration douanière avec une requête, un échantillon du produit et un dossier technique
- La décision est rendue dans un délai de **90 jours** à compter de la demande
- Le RTC est valable **6 ans** à compter de sa date d'émission
- Les décisions sont contraignantes pour toutes les administrations douanières
- Le RTC peut être annulé en cas d'informations erronées ou incomplètes

### Étapes pratiques de classification

Les étapes à suivre pour déterminer le bon code SH :

1. Définissez précisément le produit : composition, fonction, usage prévu, dimensions
2. Identifiez le chapitre concerné (catégorie générale du produit)
3. Lisez attentivement les notes de chapitre (produits inclus et exclus)
4. Examinez les libellés des positions
5. Appliquez les règles d'interprétation pour sélectionner la sous-position la plus appropriée
6. Vérifiez les subdivisions nationales
7. Si nécessaire, déposez une demande de RTC

## Détermination des taux de droits

### Types de droits de douane

Les types de droits applicables en fonction du code SH sont les suivants :

- **Droits ad valorem** : calculés en pourcentage de la valeur en douane des marchandises. Par exemple, 10 % de droits de douane
- **Droits spécifiques** : montant fixe par poids, volume ou unité. Par exemple, 0,50 EUR par kg
- **Droits mixtes** : application combinée de droits ad valorem et spécifiques
- **Droit de douane additionnel** : droit de protection supplémentaire appliqué aux produits agricoles
- **Droit antidumping** : droit supplémentaire perçu contre les pratiques de prix déloyales
- **Droit compensateur** : droit appliqué contre les produits subventionnés

### Applications tarifaires préférentielles

Dans le cadre des accords de libre-échange signés par la Turquie et de l'Union douanière, des droits de douane réduits ou nuls sont appliqués aux produits répondant à certaines conditions d'origine :

- **Union douanière UE** : droits de douane nuls sur les produits industriels (avec certificat A.TR)
- **Pays de l'AELE** : tarifs réduits dans le cadre de l'ALE (avec certificat EUR.1)
- **Corée du Sud, Singapour, Malaisie** : tarifs préférentiels dans le cadre des ALE
- **Pays du G-20 et D-8** : divers régimes préférentiels

## Erreurs de classification et sanctions

### Erreurs fréquentes

Les erreurs les plus courantes en matière de classification douanière :

1. **Confondre des produits similaires** : par exemple, le parfum et l'eau de toilette sont classés dans des positions différentes
2. **Ne pas tenir compte de la composition** : les vêtements en coton et en synthétique ont des codes SH différents
3. **Évaluer incorrectement la fonction** : pour les appareils multifonctions, la fonction principale est déterminante
4. **Mal classifier les ensembles et assortiments** : les assortiments sont classés selon l'élément qui confère le caractère essentiel
5. **Ne pas lire les notes de chapitre** : de nombreux chapitres contiennent des listes d'inclusions/exclusions
6. **Utiliser des codes obsolètes** : le tarif douanier est mis à jour chaque année

### Sanctions

Les conséquences d'une mauvaise classification peuvent être graves :

- **Différentiel de droits** : les droits de douane insuffisamment payés sont perçus avec les intérêts
- **Amende** : conformément à la législation douanière, une amende pouvant aller jusqu'à **3 fois** le différentiel de droits peut être appliquée
- **Saisie des marchandises** : en cas de fausse déclaration intentionnelle, les marchandises peuvent être saisies
- **Accusation de contrebande** : une fausse déclaration intentionnelle et systématique peut constituer une infraction pénale au sens de la législation anti-contrebande
- **Perte de réputation** : la dégradation de votre profil de risque douanier entraîne des contrôles plus stricts pour vos opérations futures

## Exemples pratiques

### Exemple 1 : Classification d'un produit textile

Une entreprise souhaite exporter des chemises pour hommes composées de 65 % coton et 35 % polyester. Processus de classification :

- Chapitre 62 : Vêtements non tricotés (tissu tissé)
- Position 6205 : Chemises pour hommes
- Sous-position 6205.20 : En coton (car le coton est prédominant)
- Code SH : 6205.20.00.00.00

### Exemple 2 : Classification d'un appareil électronique

Pour un appareil photo numérique doté à la fois d'une fonction d'enregistrement vidéo et de prise de photos :

- Chapitre 85 : Appareils électriques
- Position 8525 : Appareils émetteurs, caméras de télévision, appareils photo numériques
- Si la fonction principale est l'enregistrement vidéo : 8525.80 (caméras de télévision)
- Si la fonction principale est la prise de photos : 9006 (appareils photographiques — chapitre différent !)

### Exemple 3 : Classification d'un produit alimentaire

Noisettes enrobées de chocolat :

- Note du Chapitre 18 : Préparations alimentaires contenant du cacao
- Position 1806 : Chocolat et préparations alimentaires contenant du cacao
- Sous-position 1806.90 : Autres produits chocolatés
- Le chocolat est considéré comme conférant le caractère essentiel (Règle 3b)

## Sources de recherche des codes SH

Ressources disponibles pour déterminer le bon code SH :

- **Système de consultation du ministère du Commerce** : système officiel de requête en ligne
- **Système BILGE** : système intégré utilisé pour l'établissement des déclarations en douane
- **Base de données SH de l'OMD** : recherche de codes SH au niveau international
- **Base de données TARIC de l'UE** : base de données des tarifs et mesures commerciales de l'UE
- **Laboratoires douaniers** : appui à la classification par l'analyse chimique et physique du produit
